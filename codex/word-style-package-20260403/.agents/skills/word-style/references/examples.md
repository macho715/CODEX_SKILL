# Examples

## Example 1: archaic approval language

- 원문: 본 건을 추진코자 하오니 검토후 승인하여 주시기 바랍니다.
- 수정문: 본 건을 추진하고자 하오니 검토 후 승인 부탁드립니다.
- 메모: 의미를 바꾸지 않고 오래된 상신문 표현과 띄어쓰기를 정리했다.

## Example 2: noun-stack cleanup

- 원문: 현실적 예상 금액을 산출, 반영하여 예산 조정 추진 예정입니다.
- 수정문: 현실적인 금액을 산정해 반영하고 예산 조정을 추진할 예정입니다.
- 메모: 명사 나열을 풀어 문장 흐름만 자연스럽게 정리했다.

## Example 3: structure preserved, numbers preserved

- 원문: 계약기간은 2026.04.01 ~ 2026.12.31 이며, 총 수량은 120 EA 입니다.
- 수정문: 계약기간은 2026.04.01 ~ 2026.12.31이며, 총 수량은 120 EA입니다.
- 메모: 숫자, 날짜, 단위는 그대로 두고 조사와 띄어쓰기만 정리했다.

## Example 4: patch list mode

```md
[Page 3]
- 원문: 현시점 운영중 설비에 대한 변경전 / 변경후 비교 검토가 필요합니다.
- 수정안: 현 시점 운영 중 설비에 대한 변경 전 / 변경 후 비교 검토가 필요합니다.
- 사유: 띄어쓰기 정리
```

## Example 5: inspection memo

```md
[검수 메모]
- Amendment 번호는 계약정보라서 원문 그대로 유지
- 업체명 표기는 오타로 보일 수 있으나 근거가 없어 유지
```
