# Word Style Reference

## 6대 원칙
1. 스타일 우선
2. 구조 우선
3. 간격은 문단 속성으로 제어
4. 표는 데이터만 담기
5. 접근성 기본 내재화
6. 템플릿으로 고정

## 핵심 해석
- 문서를 “예쁘게 보이게” 만드는 것이 아니라 “반복 가능한 구조”로 정리하는 것이 목적이다.
- 본문 기본 정렬은 좌측 정렬이다.
- 표는 단순한 구조와 헤더 반복을 중심으로 설계한다.
- Accessibility Checker는 최종 품질 게이트다.

## 실무 적용
- Title / Heading / Normal / Caption / Table Text를 분리한다.
- 문단 간격은 Before/After 개념으로 관리한다.
- 빈 줄 반복, 장식용 색상, 과도한 밑줄은 제거한다.
- 숫자/통화/날짜/단위를 전체 문서에서 통일한다.
