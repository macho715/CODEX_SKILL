# Font and Layout Policy

## Font mapping
- Latin / English: Aptos
- East Asian / Korean: 맑은 고딕

## Alignment
- 기본 본문은 좌측 정렬
- 제목도 기본은 좌측 정렬
- center는 표지형/특수 요소에만 제한

## Spacing
- 기본 본문은 10.50pt + 1.30 line + after 6pt를 기준선으로 삼는다.
- 장문 가이드면 1.30~1.50까지 허용한다.
- 공백 줄 여러 개로 간격을 만들지 않는다.

## AMBER conditions
다음은 text-only 환경에서 확정하지 않는다.
- 정확한 page break
- run-level font 적용 여부
- style pane 설정값
- section break
