# KR Pro Style Set

## Global font mapping
- Latin / English: `Aptos`
- East Asian / Korean: `맑은 고딕`

Word 스타일에서는 가능하면 **Latin font = Aptos / East Asian font = 맑은 고딕**으로 설정한다.

## 실제 적용 규칙
### 제목 1
- 문서 첫 페이지 최상위 제목에만 사용
- 예: `HVDC Project Logistics Monthly Report`

### 제목 2
- 장 구분용
- 예: `1. 개요`, `2. 주요 이슈`, `3. 비용 검토`

### 제목 3
- 세부 항목용
- 예: `2.1 항만 운영`, `2.2 내륙 운송`

### 본문
- 한 문단은 한 메시지
- 3~4줄 이상 장문은 분리 검토
- 좌측 정렬 유지
- 줄간격은 1.30~1.50 권장

### 표 헤더
- Bold
- 중앙 정렬
- 연한 회색 음영 1회 사용 권장
- 페이지 넘어가면 헤더 반복

### 표 본문
- 텍스트: 좌측 정렬
- 숫자/금액: 우측 정렬
- 날짜: 중앙 또는 우측 정렬
- 셀 병합 최소화
- 단위는 열 또는 헤더에 명시

### 주석/비고
- 본문보다 작게
- 회색 계열 허용
- 법적/계약 리스크면 작은 글씨로 숨기지 않고 본문으로 올림

## 문서 기본
```text
[문서 기본]
영문: Aptos
한글: 맑은 고딕
본문: 10.50pt
정렬: 좌측 정렬
줄간격: 1.30
문단 뒤: 6pt
```

## 최종 추천
가장 기본적인 내부 보고서 기준:
- 한글 본문: 맑은 고딕 10.50pt
- 영문/Latin: Aptos
- 제목2: 13.00pt Bold
- 표 본문: 9.50pt
- 줄간격: 1.30
- 정렬: 좌측 정렬
- 표 숫자: 우측 정렬
