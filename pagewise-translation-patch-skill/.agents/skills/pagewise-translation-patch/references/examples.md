# Examples

## Example 1 — Literal English cleanup
[Page 2]
- 기존: “The cargo inward to site is pending.”
- 변경: “Site delivery of the cargo is pending.”
- 사유: 번역투 제거 / 문체 보정

## Example 2 — Awkward Korean noun-stack cleanup
[Page 5]
- 기존: “선적 서류 준비를 진행 중입니다.”
- 변경: “선적 서류를 준비하고 있습니다.”
- 사유: 번역투 제거 / 문체 보정

## Example 3 — Unsupported quantitative wording removal
[Page 8]
- 기존: “There are several delays in marine transport.”
- 변경: “Schedule delays were identified in marine transport.”
- 사유: 수치 불명확 / 문체 보정

## Example 4 — Terminology consistency
[Page 11]
- 기존: “해상 운송 스케줄”
- 변경: “해상 운송 일정”
- 사유: 용어 통일

## Example 5 — DNT handling
사용자가 아래를 DNT로 지정한 경우:
- Incoterms
- bill of lading number
- vessel name
- project code
- package ID

해당 항목은 승인 없이 번역하거나 표기를 바꾸지 않는다.
