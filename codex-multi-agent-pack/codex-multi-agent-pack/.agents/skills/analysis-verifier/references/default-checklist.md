# Default Verification Checklist

## Required checks

### 1. Planner-first structure
PASS if the draft explicitly starts with a planner or plan section that defines the task, lanes, and success criteria.
FAIL if specialist content appears before planning.

### 2. Specialist coverage
PASS if every required specialist lane appears with at least one scoped response.
AMBER if a lane appears but overlaps heavily or lacks scope boundaries.
FAIL if any required lane is missing.

### 3. PASS / FAIL separation
PASS if the verifier clearly distinguishes what passed and what failed.
FAIL if the verifier only gives a blended narrative.

### 4. Replan limit
PASS if the workflow allows zero or one replan.
FAIL if the workflow loops without a hard stop.

### 5. Structured reproducibility
PASS if another reviewer can reproduce the reasoning path from sections, criteria, and findings.
AMBER if structure exists but criteria are vague.
FAIL if the result is only a free-form summary.
