# Weighting Guide

## Recommended defaults
- Impact: 35
- Feasibility: 25
- Risk: 20
- Time-to-value: 20

## Good practice
- Keep the weight total at 100.
- Prefer 4-6 criteria.
- Keep the score scale fixed across options.
- Convert every criterion to a "higher is better" scale before scoring totals.
- For risk, score lower risk as a higher desirability score unless the user explicitly wants raw risk exposure shown.

## Sensitivity note
If the top option wins by less than 5.00 points, report the result as close and mention which criterion would most likely flip the ranking.
